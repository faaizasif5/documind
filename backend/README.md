# DocuMind Backend

FastAPI backend for a RAG document Q&A application.

## Local Setup

```powershell
cd backend
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install -e ".[dev]"
Copy-Item .env.example .env
fastapi dev app/main.py
```

The API health endpoint is available at:

```text
GET /api/v1/healthz
```

## Database Migrations

Migrations use Alembic over the Supabase **session pooler** (`DATABASE_MIGRATION_URL`,
port 5432). The app runtime uses the **transaction pooler** (`DATABASE_URL`, port 6543).

```powershell
# apply all migrations
alembic upgrade head

# roll back the latest migration
alembic downgrade -1

# create a new autogenerated migration after model changes
alembic revision --autogenerate -m "describe change"
```
